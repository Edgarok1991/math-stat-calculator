# Макеты визуализации MathCalc

Здесь лежат **40 PNG-макетов** (1200×675), перечисленных в [`DESIGN_PROPOSALS.md`](../DESIGN_PROPOSALS.md).

## Путь

**Абсолютный путь (на вашем Mac):**  
`/Users/edgar/Desktop/Project/math-stat-calculator/assets/`

**Относительно корня репозитория:**  
`assets/`

## Пересоздать файлы

После правок палитр в `scripts/generate-mockups.js`:

```bash
npm run generate-mockups
```

Требуется пакет `sharp` (devDependency).
