## Вариант 1: «Академический / научный»

**Идея:** Строгий, «учебниковый» стиль — как у научных публикаций и математических редакторов.

| Элемент | Изменение |
|---------|-----------|
| **Шрифт** | Georgia / Lora / Source Serif Pro — классическая читаемость |
| **Цвета** | Тёплые нейтральные: кремовый фон (#faf8f5), тёмно-коричневый текст (#2c1810), акцент — тёмно-синий (#1e3a5f) |
| **Карточки** | Без градиентов, тонкие рамки, лёгкие тени |
| **Формулы** | Более крупный и контрастный шрифт для математики |
| **Навигация** | Подчёркивание вместо заливки, минималистичный хедер |

**Плюсы:** Серьёзный вид, удобно для длительной работы.  
**Минусы:** Может казаться скучным.

## Рекомендуемые улучшения (под любой вариант)

1. **Типографика**
   - Увеличить контраст заголовков (font-weight, размер).
   - Добавить чёткую иерархию (h1, h2, h3).

2. **Формы ввода**
   - Добавить `autocomplete` и `aria-label` для доступности.
   - Улучшить визуальную обратную связь (focus, ошибки).

3. **Графики**
   - Единая цветовая палитра для всех графиков.
   - Поддержка тёмной темы в Plotly.

4. **Мобильная версия**
   - Проверить отступы и размеры кнопок на маленьких экранах.
   - Упростить навигацию на мобильных.

5. **Анимации**
   - Сократить длительность (0.2–0.3 с).
   - Добавить `prefers-reduced-motion` для пользователей с ограничениями.

## Быстрый старт

Чтобы применить один из вариантов:

1. Выберите вариант (1–40).
2. Обновите `globals.css` (цвета, переменные).
3. При необходимости смените шрифт в `layout.tsx`.
4. Подправьте компоненты (Header, кнопки, карточки) под выбранный стиль.

Могу помочь с реализацией любого варианта пошагово.
