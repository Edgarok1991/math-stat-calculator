# Математический калькулятор для статистических вычислений

Полнофункциональное веб-приложение для выполнения различных математических и статистических вычислений.

## 🚀 Возможности

### 📊 Статистические расчеты
- **ANOVA** - дисперсионный анализ
- **Кластерный анализ** - иерархическая кластеризация с подробными таблицами
  - Метод ближнего соседа (Single Linkage)
  - Метод дальнего соседа (Complete Linkage)
  - Метод среднего (Average Linkage)
  - K-means кластеризация
  - Построение дендрограммы
  - Пошаговое отображение матриц расстояний
- **Регрессионный анализ** - полиномиальная регрессия

### 🔢 Матричные операции
- **Метод Гаусса** - решение систем линейных уравнений
- **Обратная матрица** - вычисление обратной матрицы
- **Определитель** - вычисление определителя матрицы
- **Умножение матриц** - операции с матрицами

### 📈 Математический анализ
- **Производные** - вычисление производных функций
- **Интегралы** - вычисление определенных интегралов

## 📦 Установка и запуск

### Предварительные требования
- Node.js 18+ 
- npm или yarn

### Установка зависимостей
```bash
# Установка зависимостей для всего проекта
npm install

# Или для каждой части отдельно
cd frontend && npm install
cd backend && npm install
```

### Запуск в режиме разработки

#### Вариант 1: Автоматический запуск
```bash
# Запуск frontend и backend одновременно
npm run dev
```

#### Вариант 2: Ручной запуск
```bash
# Терминал 1: Backend (порт 3001)
cd backend && npm run start:dev

# Терминал 2: Frontend (порт 3000)
cd frontend && npm run dev
```

### Сборка для продакшена
```bash
# Сборка backend
cd backend && npm run build

# Сборка frontend
cd frontend && npm run build
```

## 🎯 Использование

1. **Откройте браузер** и перейдите на `http://localhost:3000`
2. **Выберите тип вычислений** из главного меню
3. **Введите данные** в соответствующие поля
4. **Нажмите "Рассчитать"** для получения результата

## 🔧 Особенности

### Точные вычисления
- Все числа отображаются в виде **дробей** вместо десятичных
- Высокая точность вычислений
- Пошаговые решения с подробными объяснениями

### Кластерный анализ
- Детальные таблицы расстояний на каждом шаге
- Цветовое выделение объединяемых кластеров
- Построение дендрограммы
- Формат ввода данных как на сайте [axd.semestr.ru](https://axd.semestr.ru/upr/cluster.php)
- Подробная инструкция: см. [CLUSTERING_GUIDE.md](CLUSTERING_GUIDE.md)

### Пользовательский интерфейс
- Адаптивный дизайн
- Темная/светлая тема
- Интуитивно понятный интерфейс
- Валидация данных в реальном времени

## 📝 Примеры использования

### Решение системы уравнений
```
2x + 3y = 7
x - y = 1
```
Результат: x = 2, y = 1

### ANOVA анализ
Введите данные групп и получите:
- F-статистику
- p-значение
- Выводы о значимости различий

### Кластерный анализ
```
Исходные данные:
2, 8
4, 10
5, 7
12, 6
14, 6
15, 4

Метод: Ближний сосед
Кластеры: 2
```

Результат:
- **S(1,2,3)** - объекты [1, 2, 3]
- **S(4,5,6)** - объекты [4, 5, 6]
- Расстояние: P = 7.07
- Пошаговые таблицы + дендрограмма
