## 🎨 Визуализация

### Цветовая схема

| Элемент | Цвет | Назначение |
|---------|------|------------|
| Заголовок шага | 🔵 Синий | Номер и название шага |
| Объединяемые кластеры | 🟡 Жёлтый | Строки/столбцы в таблице |
| Минимальное расстояние | 🔴 Красный | Ячейка с минимумом |
| Пояснения | 🟢 Зелёный | Описания и заключение |

### Компоненты результата

1. **Краткие результаты** (справа от формы)
   - Метод кластеризации
   - Количество кластеров
   - Финальное расстояние
   - Состав кластеров

2. **Исходные данные** (таблица)
   - Все объекты с координатами
   - Нумерация

3. **Пошаговое решение**
   - Матрицы расстояний на каждом шаге
   - Выделение минимумов
   - Описания объединений
   - Текущие кластеры

4. **Дендрограмма**
   - Иерархическая структура
   - Расстояния объединения
   - Подписи объектов

5. **Заключение**
   - Итоговая сводка
   - Финальные кластеры
   - Расстояние между ними

## 🔧 API

### Эндпоинт

```
POST http://localhost:3001/clustering/calculate
```

### Запрос

```json
{
  "points": [
    [2, 8],
    [4, 10],
    [5, 7],
    [12, 6],
    [14, 6],
    [15, 4]
  ],
  "k": 2,
  "method": "single"
}
```

```json
{

  "centroids": [[3.67, 8.33], [13.67, 5.33]],

  "inertia": 58.67,
  "method": "single",
  "finalDistance": 7.07,
  "initialDistanceMatrix": [...],
  "dendrogramData": {...},
  "steps": [...]
}
```
