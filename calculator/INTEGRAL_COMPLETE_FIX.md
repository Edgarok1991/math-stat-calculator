### Базовые

| Функция | Интеграл |
|---------|----------|
| x | x²/2 + C |
| x² | x³/3 + C |
| x³ | x⁴/4 + C |
| 1/x | ln\|x\| + C |

### Тригонометрические

| Функция | Интеграл |
|---------|----------|
| sin(x) | -cos(x) + C |
| cos(x) | sin(x) + C |
| tan(x) | -ln\|cos(x)\| + C |

### Показательные

| Функция | Интеграл |
|---------|----------|
| eˣ | eˣ + C |
| ln(x) | x·ln(x) - x + C |

| x·cos(x) | cos(x) + x·sin(x) + C |
