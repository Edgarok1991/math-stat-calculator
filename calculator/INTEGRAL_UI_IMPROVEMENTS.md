## 🔧 Технические детали

### Новый компонент IntegralSymbol

**Файл:** `frontend/src/components/UI/IntegralSymbol.tsx`

**Возможности:**
- Два типа: `indefinite` | `definite`
- Три размера: `small` | `medium` | `large`
- Единый стиль отображения

**Использование:**
```tsx
<IntegralSymbol type="definite" size="medium" />
```

**Размеры:**

| Размер | Интеграл | Пределы | Применение |
|--------|----------|---------|------------|
| small | text-lg | 0.5rem | Переключатель, подсказки |
| medium | text-2xl | 0.65rem | Кнопки, заголовки |
| large | text-3xl | text-xs | Крупные заголовки |

### Компонент TextWithFractions

**Уже существует:** `frontend/src/components/UI/TextWithFractions.tsx`

**Функционал:**
- Автоматически находит дроби в тексте (формат: число/число)
- Преобразует в математические дроби
- Поддерживает отрицательные числа

**Примеры:**
```
"1/3" →     1
           ―
            3

"1/3*x^3 + C" →  1
                ―*x³ + C
                 3
```

## 📊 Примеры отображения

### Неопределённый интеграл

**Результат:**
```
∫ Первообразная:

 1          
―·x³ + C    
 3          
```

**Пошаговое решение:**
```
1. Исходная функция: x²
2. Применяем правила интегрирования
3. ∫(xⁿ)dx = xⁿ⁺¹/(n+1) + C
4. Результат: ―·x³ + C
               3
```

### Определённый интеграл

**Результат:**
```
  b
  ∫  Численное значение:
  a

0.333... или  1
             ―
              3
```

**Пошаговое решение:**
```
1. Исходная функция: x²
2. Находим первообразную
3. F(x) = ―·x³ + C
           3
4. Вычисляем F(1) - F(0)
5. F(1) = 0.333...
6. Результат: 0.333...
```
