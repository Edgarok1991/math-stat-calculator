# ∫ Интегралы - Краткая справка

## ✅ Что добавлено

Теперь можно выбрать тип интеграла:
- 📝 **Неопределённый** - ∫ f(x) dx = F(x) + C
- 📊 **Определённый** - ∫ᵇₐ f(x) dx = число

---

## 🎯 Как использовать

### Неопределённый интеграл

1. **Выберите:** Неопределённый интеграл
2. **Введите:** функцию (например, x^2)
3. **Результат:** первообразная с константой C

**Пример:**
```
Функция: x^2
Результат: (x³/3) + C
```

### Определённый интеграл

1. **Выберите:** Определённый интеграл
2. **Введите:** функцию (например, x^2)
3. **Укажите:** пределы (a = 0, b = 1)
4. **Результат:** число (площадь)

**Пример:**
```
Функция: x^2
Пределы: от 0 до 1
Результат: 0.333... (1/3)
```

---

## 🎨 Интерфейс

### Переключатель

```
[●] Неопределённый  [ ] Определённый
    ∫ f(x) dx           ∫ᵇₐ f(x) dx
```

### Поля

**Неопределённый:**
- Функция ✓
- Переменная ✓
- Пределы ✗ (скрыты)

**Определённый:**
- Функция ✓
- Переменная ✓
- Нижний предел ✓*
- Верхний предел ✓*

---

## 📊 Примеры

| Тип | Функция | Пределы | Результат |
|-----|---------|---------|-----------|
| Неопределённый | x² | - | (x³/3) + C |
| Неопределённый | sin(x) | - | -cos(x) + C |
| Определённый | x² | [0, 1] | 1/3 ≈ 0.333 |
| Определённый | sin(x) | [0, π] | 2 |
