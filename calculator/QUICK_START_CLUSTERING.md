# 🚀 Быстрый старт - Кластерный анализ

## Запуск приложения

### 1. Запустить Backend и Frontend

```bash
# В одном терминале (из корневой папки calculator)
cd backend && npm run start:dev

# В другом терминале (из корневой папки calculator)  
cd frontend && npm run dev
```

### 2. Открыть в браузере

```
http://localhost:3000/clustering
```

## 🎯 Использование

### Шаг 1: Загрузить пример

Нажмите кнопку **"Загрузить пример с сайта"** - автоматически заполнятся данные:

```
2, 8
4, 10
5, 7
12, 6
14, 6
15, 4
```

### Шаг 2: Выбрать параметры

- **Количество кластеров**: 2
- **Метод**: 🔵 Ближний сосед (Single Linkage)

### Шаг 3: Рассчитать

Нажмите кнопку **"Выполнить кластеризацию"**

### Шаг 4: Изучить результаты

Вы получите:

1. **Краткие результаты** (справа)
   - Метод кластеризации
   - Количество кластеров
   - Расстояние между кластерами: **P = 7.07**

2. **Исходные данные** (внизу)
   - Таблица всех объектов

3. **Пошаговое решение**
   - 📊 Матрицы расстояний на каждом шаге
   - 🎯 Выделение минимальных расстояний
   - 📝 Подробные объяснения

4. **Дендрограмма**
   - Визуализация иерархии объединения
   - Расстояния на каждом уровне

5. **Заключение**
   - Итоговая сводка

## 💡 Советы

### Для детального решения
✅ Используйте **иерархические методы**:
- 🔵 Ближний сосед
- 🔴 Дальний сосед  
- 🟡 Средний

❌ K-means не показывает пошаговые таблицы

### Рекомендуемые данные
- **4-10 объектов** - оптимально для наглядности
- **2-3 признака** - удобно для визуализации
