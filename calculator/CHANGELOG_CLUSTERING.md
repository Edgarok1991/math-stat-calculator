## 🔧 Технические детали

### Алгоритмы

**Иерархическая кластеризация:**
1. Инициализация - каждый объект = отдельный кластер
2. Расчёт матрицы расстояний между всеми объектами
3. Поиск пары с минимальным расстоянием
4. Объединение найденной пары
5. Перерасчёт матрицы расстояний
6. Повторение шагов 3-5 до достижения нужного количества кластеров

**Методы расчёта расстояний между кластерами:**
- **Single Linkage (ближний сосед):** `min{d(a,b) : a ∈ A, b ∈ B}`
- **Complete Linkage (дальний сосед):** `max{d(a,b) : a ∈ A, b ∈ B}`
- **Average Linkage (средний):** `avg{d(a,b) : a ∈ A, b ∈ B}`

**Евклидово расстояние:**
```
d(i,j) = √[(x₁ᵢ - x₁ⱼ)² + (x₂ᵢ - x₂ⱼ)² + ...]
```

### API

**Эндпоинт:** `POST /clustering/calculate`

**Запрос:**
```json
{
  "points": [[2, 8], [4, 10], [5, 7], [12, 6], [14, 6], [15, 4]],
  "k": 2,
  "method": "single"
}
```

**Ответ:**
```json
{
  "clusters": [[0, 1, 2], [3, 4, 5]],
  "centroids": [[3.67, 8.33], [13.67, 5.33]],
  "labels": [0, 0, 0, 1, 1, 1],
  "inertia": 58.67,
  "method": "single",
  "finalDistance": 7.07,
  "initialDistanceMatrix": [...],
  "dendrogramData": {...},
  "steps": [...]
}
```
