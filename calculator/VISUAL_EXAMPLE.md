# 🎨 Визуальный пример работы калькулятора

## 📊 Пошаговое решение кластерного анализа

### Исходные данные

```
┌──────────┬────┬────┐
│ № объекта│ x₁ │ x₂ │
├──────────┼────┼────┤
│    1     │ 2  │ 8  │
│    2     │ 4  │ 10 │
│    3     │ 5  │ 7  │
│    4     │ 12 │ 6  │
│    5     │ 14 │ 6  │
│    6     │ 15 │ 4  │
└──────────┴────┴────┘
```

---

## Шаг 0: Инициализация

**Описание:** Создана начальная матрица расстояний для 6 объектов. Каждый объект является отдельным кластером.

### Матрица расстояний

```
┌─────┬──────┬──────┬──────┬──────┬──────┬──────┐
│ №п/п│ S(1) │ S(2) │ S(3) │ S(4) │ S(5) │ S(6) │
├─────┼──────┼──────┼──────┼──────┼──────┼──────┤
│ S(1)│ 0.00 │ 2.83 │ 3.16 │10.20 │12.17 │13.60 │
│ S(2)│ 2.83 │ 0.00 │ 3.16 │ 8.94 │10.77 │12.53 │
│ S(3)│ 3.16 │ 3.16 │ 0.00 │ 7.07 │ 9.06 │10.44 │
│ S(4)│10.20 │ 8.94 │ 7.07 │ 0.00 │ 2.00 │ 3.61 │ ← минимум
│ S(5)│12.17 │10.77 │ 9.06 │ 2.00 │ 0.00 │ 2.24 │
│ S(6)│13.60 │12.53 │10.44 │ 3.61 │ 2.24 │ 0.00 │
└─────┴──────┴──────┴──────┴──────┴──────┴──────┘
```

**Текущие кластеры (6):**
```
S(1)  S(2)  S(3)  S(4)  S(5)  S(6)
```

---

## Шаг 1: Объединение кластеров методом ближнего соседа

**⚡ Минимальное расстояние:** P₄,₅ = 2.00

**📝 Объяснение:** Из матрицы расстояний следует, что объекты S(4) и S(5) наиболее близки (P = 2.00) и поэтому объединяются в один кластер.

### Матрица расстояний

```
┌──────┬──────┬──────┬──────┬────────┬──────┐
│  №п/п│ S(1) │ S(2) │ S(3) │ S(4,5) │ S(6) │
├──────┼──────┼──────┼──────┼────────┼──────┤
│ S(1) │ 0.00 │ 2.83 │ 3.16 │ 10.20  │13.60 │
│ S(2) │ 2.83 │ 0.00 │ 3.16 │  8.94  │12.53 │
│ S(3) │ 3.16 │ 3.16 │ 0.00 │  7.07  │10.44 │
│S(4,5)│10.20 │ 8.94 │ 7.07 │  0.00  │ 2.24 │ ← минимум
│ S(6) │13.60 │12.53 │10.44 │  2.24  │ 0.00 │
└──────┴──────┴──────┴──────┴────────┴──────┘
```

**Текущие кластеры (5):**
```
S(1)  S(2)  S(3)  S(4,5)  S(6)
```

---

## Шаг 2: Объединение кластеров методом ближнего соседа

**⚡ Минимальное расстояние:** P₄₅,₆ = 2.24

**📝 Объяснение:** Из матрицы расстояний следует, что объекты S(4,5) и S(6) наиболее близки (P = 2.24) и поэтому объединяются в один кластер.

### Матрица расстояний

```
┌────────┬──────┬──────┬──────┬──────────┐
│   №п/п │ S(1) │ S(2) │ S(3) │ S(4,5,6) │
├────────┼──────┼──────┼──────┼──────────┤
│  S(1)  │ 0.00 │ 2.83 │ 3.16 │  10.20   │
│  S(2)  │ 2.83 │ 0.00 │ 3.16 │   8.94   │ ← минимум
│  S(3)  │ 3.16 │ 3.16 │ 0.00 │   7.07   │
│S(4,5,6)│10.20 │ 8.94 │ 7.07 │   0.00   │
└────────┴──────┴──────┴──────┴──────────┘
```

**Текущие кластеры (4):**
```
S(1)  S(2)  S(3)  S(4,5,6)
```

---

## Шаг 3: Объединение кластеров методом ближнего соседа

**⚡ Минимальное расстояние:** P₁,₂ = 2.83

**📝 Объяснение:** Из матрицы расстояний следует, что объекты S(1) и S(2) наиболее близки (P = 2.83) и поэтому объединяются в один кластер.

### Матрица расстояний

```
┌──────────┬────────┬──────┬──────────┐
│   №п/п   │ S(1,2) │ S(3) │ S(4,5,6) │
├──────────┼────────┼──────┼──────────┤
│  S(1,2)  │  0.00  │ 3.16 │   8.94   │ ← минимум
│   S(3)   │  3.16  │ 0.00 │   7.07   │
│ S(4,5,6) │  8.94  │ 7.07 │   0.00   │
└──────────┴────────┴──────┴──────────┘
```

**Текущие кластеры (3):**
```
S(1,2)  S(3)  S(4,5,6)
```

---

## Шаг 4: Объединение кластеров методом ближнего соседа

**⚡ Минимальное расстояние:** P₁₂,₃ = 3.16

**📝 Объяснение:** Из матрицы расстояний следует, что объекты S(1,2) и S(3) наиболее близки (P = 3.16) и поэтому объединяются в один кластер.

### Матрица расстояний

```
┌────────────┬────────────┬──────────┐
│    №п/п    │  S(1,2,3)  │ S(4,5,6) │
├────────────┼────────────┼──────────┤
│  S(1,2,3)  │    0.00    │   7.07   │ ← финальное расстояние
│  S(4,5,6)  │    7.07    │   0.00   │
└────────────┴────────────┴──────────┘
```

**Текущие кластеры (2):**
```
S(1,2,3)  S(4,5,6)
```

## 🎯 Итоговые результаты

### Кластеры

```
┌───────────────┬──────────────────────────────┐
│   Кластер     │         Объекты              │
├───────────────┼──────────────────────────────┤
│   S(1,2,3)    │  [1, 2, 3] - левая группа    │
│   S(4,5,6)    │  [4, 5, 6] - правая группа   │
└───────────────┴──────────────────────────────┘
```

### Метрики

```
📊 Метод: Ближний сосед (Single Linkage)
📈 Количество кластеров: 2
📏 Расстояние между кластерами: P = 7.07
```

---

## 🌲 Дендрограмма

```
         ┌──────────────────────────── 7.07 ────────────────┐
         │                                                   │
    ┌── 3.16 ──┐                                       ┌── 2.24 ──┐
    │          │                                       │           │
┌─ 2.83 ─┐     │                                   ┌─ 2.00 ─┐     │
│         │     │                                   │        │     │
│         │     │                                   │        │     │
1         2     3                                   4        5     6
(2,8)   (4,10) (5,7)                              (12,6)  (14,6) (15,4)
```

**Легенда:**
- Числа на линиях = расстояние объединения
- Чем выше линия, тем больше различие между кластерами

---

## 📝 Заключение

> Таким образом, при проведении кластерного анализа по принципу **"ближнего соседа"** 
> получили **2 кластера**, расстояние между которыми равно **P = 7.07**.
> 
> Результаты иерархической классификации объектов представлены выше в виде дендрограммы.
