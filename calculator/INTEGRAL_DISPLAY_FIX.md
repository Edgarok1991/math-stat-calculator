## 📍 Где изменено

### 1. Переключатель типа интеграла

**Определённый интеграл:**
```
  b
  ∫  f(x) dx = F(b) - F(a)
  a
```

### 2. Кнопка вычисления

**Текст кнопки:**
```
  b
  ∫  Вычислить определённый интеграл
  a
```

### 3. Заголовок результата

**Когда показан результат:**
```
  b
  ∫  Численное значение: 0.333...
  a
```

## 🔧 Технические детали

### Реализация

**HTML/CSS структура:**
```tsx
<span className="relative inline-flex flex-col items-center">
  <span style={{ fontSize: '0.6rem' }}>b</span>
  <span style={{ fontSize: '1.3rem' }}>∫</span>
  <span style={{ fontSize: '0.6rem' }}>a</span>
</span>
```

**Стили:**
- `flex-col` - вертикальное расположение
- `items-center` - центрирование
- `fontSize: '0.6rem'` - маленькие буквы для пределов
- `fontSize: '1.3rem'` - большой знак интеграла
