# 🏠 Страница авторизации как главная

## ✅ Что изменено

Теперь **страница авторизации** - это **начальная страница** приложения!

---

## 🎯 Поведение

### Для неавторизованных пользователей

**Открывают:** `http://localhost:3000`

**Видят:** Автоматический редирект на `/auth`

**Результат:** Страница входа/регистрации

### Для авторизованных пользователей

**Открывают:** `http://localhost:3000`

**Видят:** Главную страницу с калькуляторами

**Результат:** Доступ ко всем функциям

---

## 🔄 Workflow пользователя

### Первый визит

```
1. Открывают http://localhost:3000
   ↓
2. Редирект → http://localhost:3000/auth
   ↓
3. Видят форму входа/регистрации
   ↓
4. Регистрируются
   ↓
5. Автоматический вход
   ↓
6. Редирект → http://localhost:3000
   ↓
7. Видят главную страницу с калькуляторами
```

### Повторный визит (авторизован)

```
1. Открывают http://localhost:3000
   ↓
2. Токен найден в localStorage
   ↓
3. Автоматический вход
   ↓
4. Сразу видят главную страницу
```

### Выход

```
1. Header → Имя пользователя → Выйти
   ↓
2. Токен удаляется
   ↓
3. Редирект → http://localhost:3000/auth
```

## 🔧 Техническая реализация

### Изменённый файл

**`frontend/src/app/page.tsx`**

**Добавлено:**
```typescript
const { isAuthenticated, isLoading } = useAuth();
const router = useRouter();

useEffect(() => {
  if (!isLoading && !isAuthenticated) {
    router.push('/auth');
  }
}, [isAuthenticated, isLoading, router]);
```

**Логика:**
1. Проверяется статус авторизации
2. Если не авторизован → редирект на /auth
3. Если авторизован → показывается главная
4. Пока загрузка → показывается спиннер

## ✅ Проверка работы

### Тест 1: Новый пользователь

1. Откройте новую вкладку инкогнито
2. Перейдите на `http://localhost:3000`

2. Редирект на `/auth`
3. При попытке открыть `/` снова редирект на `/auth`
