# Настройка отправки писем (Yandex)

Ethereal Email недоступен из вашей сети. Используйте Yandex SMTP для реальной доставки писем.

## Шаг 1: Включить доступ для почтовых программ

**Важно!** Без этого шага будет ошибка «This user does not have access rights to this service».

1. Откройте [mail.yandex.ru](https://mail.yandex.ru)
2. **Настройки** (⚙️) → **Все настройки** → **Почтовые программы**
3. Включите:
   - ☑️ **С сервера pop.yandex.ru по протоколу POP3**
   - ☑️ **От сервера imap.yandex.ru по протоколу IMAP**
   - ☑️ **Пароли приложений и OAuth-токены**
4. Нажмите **Сохранить изменения**

## Шаг 2: Создать пароль приложения

1. Откройте [id.yandex.ru](https://id.yandex.ru) → **Безопасность**
2. **Пароли приложений** → **Создать пароль**
3. Укажите название (например, «MathCalc») и скопируйте пароль

> Если пункта «Пароли приложений» нет — включите двухфакторную аутентификацию.

## Шаг 3: Настроить .env

В файле `calculator/backend/.env` раскомментируйте и заполните:

```env
SMTP_HOST=smtp.yandex.ru
SMTP_PORT=587
SMTP_USER=eaelf@yandex.ru
SMTP_PASS=скопированный-пароль-приложения
```

## Шаг 4: Перезапустить backend

```bash
cd calculator/backend
npm run start:dev
```

После этого письма активации будут приходить на указанный при регистрации email.
