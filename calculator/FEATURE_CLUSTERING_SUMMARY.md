# 🎉 Кластерный анализ - Готово!

## ✅ Что реализовано

Добавлен полнофункциональный калькулятор кластерного анализа с детальным пошаговым решением, полностью соответствующий примеру на сайте https://axd.semestr.ru/upr/cluster.php

## 🎯 Ключевые возможности

### 1. Методы кластеризации
- 🔵 **Ближний сосед (Single Linkage)** - минимальное расстояние между кластерами
- 🔴 **Дальний сосед (Complete Linkage)** - максимальное расстояние между кластерами
- 🟡 **Средний (Average Linkage)** - среднее расстояние
- ⚡ **K-means** - быстрая кластеризация

### 2. Визуализация

#### Пошаговые таблицы расстояний
- Полная матрица на каждом шаге
- Цветовое выделение объединяемых кластеров
- Подсветка минимальных расстояний

#### Дендрограмма
- Визуализация иерархии объединения
- Отображение расстояний на каждом уровне
- Интерактивная графика

#### Детальные объяснения
- Описание каждого шага
- Обоснование объединения кластеров
- Итоговое заключение

## 🚀 Как использовать

### Запуск приложения

```bash
# Терминал 1: Backend
cd calculator/backend
npm run start:dev

# Терминал 2: Frontend
cd calculator/frontend
npm run dev
```

### Открыть в браузере
```
http://localhost:3000/clustering
```

### Быстрый тест

1. Нажмите **"Загрузить пример с сайта"**
2. Нажмите **"Выполнить кластеризацию"**
3. Изучите результаты:
   - Краткие результаты справа
   - Пошаговое решение внизу
   - Дендрограмма
   - Заключение
