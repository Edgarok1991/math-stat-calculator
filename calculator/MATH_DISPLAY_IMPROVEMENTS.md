### 2. Новый компонент MathExpression

**Файл:** `frontend/src/components/UI/MathExpression.tsx`

**Функционал:**
- Преобразует степени (^) в верхние индексы
- Заменяет умножение (*) на точку (·)
- Использует TextWithFractions для дробей

**Преобразования:**
```
x^2  → x²
x^3  → x³
x^10 → x¹⁰
*    → ·
1/3  → вертикальная дробь
```

**Карта верхних индексов:**
```typescript
'0' → '⁰'
'1' → '¹'
'2' → '²'
'3' → '³'
...
'9' → '⁹'
'-' → '⁻'
'+' → '⁺'
```

## 📊 Примеры отображения

### Неопределённый интеграл

**Входная функция:**
```
x^3
```

**Результат:**
```
∫ Первообразная:

 1
―·x⁴ + C
 4
```

### Определённый интеграл

**Входная функция:**
```
x^3
Пределы: от 2 до 3
```

**Результат:**
```
 b
 ∫  Численное значение:
 a

16.25
```

**Пошаговое решение:**
```
1. Исходная функция: x³

2. Находим первообразную:

3. F(x) = ―·x⁴ + C
          4

4. Вычисляем F(3) - F(2):

5. F(3) = 20.25

6. F(2) = 4

7. Результат: 16.25
```
