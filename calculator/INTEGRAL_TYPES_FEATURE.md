# ∫ Разделение интегралов - Готово!

## ✅ Что добавлено

Добавлена возможность **выбора типа интеграла**:
- 📝 **Неопределённый интеграл** - первообразная с константой C
- 📊 **Определённый интеграл** - численное значение (площадь)

---

## 🎯 Новые возможности

### 1. Переключатель типа интеграла

**Два варианта:**

#### Неопределённый интеграл
```
∫ f(x) dx = F(x) + C
```
- Результат: первообразная функция
- Содержит константу интегрирования C
- Пределы не требуются

#### Определённый интеграл
```
∫ᵇₐ f(x) dx = F(b) - F(a)
```
- Результат: число
- Площадь под графиком от a до b
- Требуются нижний и верхний пределы

---

## 🎨 Интерфейс

### Переключатель типа

```
┌─────────────────────────────────────────────────────────┐
│ [●] Неопределённый интеграл  [ ] Определённый интеграл  │
│     ∫ f(x) dx = F(x) + C         ∫ᵇₐ f(x) dx = F(b)-F(a) │
└─────────────────────────────────────────────────────────┘
```

### Поля формы

**Неопределённый интеграл:**
- ✅ Функция f(x)
- ✅ Переменная (x)
- ❌ Пределы скрыты

**Определённый интеграл:**
- ✅ Функция f(x)
- ✅ Переменная (x)
- ✅ Нижний предел (a) - обязательно*
- ✅ Верхний предел (b) - обязательно*

---

## 🔧 Технические детали

### Обновлённая схема валидации

```typescript
const integralSchema = z.object({
  expression: z.string().min(1, 'Введите функцию'),
  variable: z.string().min(1, 'Введите переменную'),
  integralType: z.enum(['indefinite', 'definite']),
  lowerBound: z.number().optional(),
  upperBound: z.number().optional(),
}).refine((data) => {
  // Для определённого интеграла пределы обязательны
  if (data.integralType === 'definite') {
    return data.lowerBound !== undefined && 
           data.upperBound !== undefined;
  }
  return true;
}, {
  message: 'Для определённого интеграла укажите оба предела',
});
```

### State управление

```typescript
const [integralType, setIntegralType] = 
  useState<'indefinite' | 'definite'>('indefinite');
```

### Условное отображение

```typescript
{integralType === 'definite' && (
  <>
    <input name="lowerBound" required />
    <input name="upperBound" required />
  </>
)}
```

## 📋 Обновлённая инструкция

### Шаг 1: Выберите тип интеграла
- **Неопределённый:** для нахождения первообразной
- **Определённый:** для вычисления площади

### Шаг 2: Введите функцию
- Поддерживаются: x^n, sin(x), cos(x), exp(x), log(x)
- Примеры: x^2, sin(x)*cos(x), x*exp(x)

### Шаг 3: Укажите параметры
- Переменная интегрирования (обычно x)
- Для определённого: пределы a и b

### Шаг 4: Получите результат
- Первообразная или численное значение
- Пошаговое решение
- График и визуализация

## 🔄 Поведение при переключении

### С неопределённого на определённый
1. Сохраняется функция и переменная
2. Появляются поля пределов
3. Меняется текст кнопки
4. Показывается подсказка про площадь

### С определённого на неопределённый
1. Сохраняется функция и переменная
2. **Пределы очищаются автоматически**
3. Поля пределов скрываются
4. Меняется текст кнопки
5. Показывается подсказка про константу C
