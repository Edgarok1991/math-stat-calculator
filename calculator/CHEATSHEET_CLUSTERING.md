## 🎯 Методы кластеризации

| Метод | Описание | Когда использовать |
|-------|----------|-------------------|
| 🔵 **Single** | Минимальное расстояние | Вытянутые кластеры |
| 🔴 **Complete** | Максимальное расстояние | Компактные кластеры |
| 🟡 **Average** | Среднее расстояние | Универсально |
| ⚡ **K-means** | Итеративный | Быстро, без таблиц |

💡 **Для пошаговых таблиц** выбирайте Single/Complete/Average

## 🔧 API

### Эндпоинт
```
POST http://localhost:3001/clustering/calculate
```

### Минимальный запрос
```json
{
  "points": [[2, 8], [4, 10], [5, 7], [12, 6], [14, 6], [15, 4]],
  "k": 2,
  "method": "single"
}
```

### Методы
- `"single"` - ближний сосед
- `"complete"` - дальний сосед
- `"average"` - средний
- `"kmeans"` - K-means

---

## 💡 Частые вопросы

### Как получить пошаговые таблицы?
Выберите метод: Single, Complete или Average

### Почему K-means не показывает таблицы?
K-means - итеративный метод, не использует иерархическую структуру

### Как изменить количество кластеров?
Измените значение в поле "Количество кластеров"

### Формат данных не подходит?
- Каждая строка = один объект
- Координаты через запятую
- Пример: `2, 8`
