# Neue Montreal (Pangram Pangram)

Используется **Neue Montreal Medium** по всему приложению.

## Файл

Положите в эту папку:

- **`NeueMontreal-Medium.woff2`**

(при необходимости переименуйте файл из архива Pangram Pangram в это имя.)

## Где взять

- [Neue Montreal — Pangram Pangram](https://pangrampangram.com/products/neue-montreal) — пробная версия или коммерческая лицензия.

Пока файла нет в репозитории, браузер использует запасные шрифты из стека в `globals.css`.

## Опционально (жирный текст)

Если нужен настоящий Bold без синтеза, добавьте `NeueMontreal-Bold.woff2` и раскомментируйте второй `@font-face` в `src/app/globals.css`.
